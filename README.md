# MedX-web

MedX-web is a healthcare dashboard frontend We built as part of Project 1 in the 3rd year of ITE in Damascus Uni. The goal was to design a clean, modern interface for basic clinic management workflows and practice building a multi-page React app with protected routes and reusable UI components.

## Project Overview

This project simulates a small medical management system where an authenticated user can navigate between:
- Overview dashboard with key stats
- Patient directory
- Appointments page
- Medical records page
- Financials page
